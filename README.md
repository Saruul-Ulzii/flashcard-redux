# Flashcard App

Single-page flashcard application to help you study!

###Screenshots


###Getting Started

    npm install

###To Start Server

    npm run dev

Go to:

    http://localhost:8080/webpack-dev-server

###To Run Unit Tests

    npm run test

or,

    npm run test:watch
